# Enumeration: EV4BasicConstraint

The three basic-order constraint classes, identified on-chain by the
constructor index of the constraint `data`. (Swap orders use a separate,
partial-fill-capable encoding — see [DatumBuilderV4.buildSwapConstraintData](../classes/DatumBuilderV4.md#buildswapconstraintdata).)
